Linux 中的相关命令

购买平台

  • 腾讯云
  • 阿里云

Linux 系统

  • centos
  • redhat

操作命令

  • 建立链接：ssh 用户名@ip地址
  • 查看文件位置：pwd
  • 查看当前目录下的文件和文件夹：ll
    • 区分文件和文件夹：看首字母
      • d: 文件夹
      • l: 软链接
      • ~：文件
  • 创建文件夹：mkdir 目录名
  • 创建文件：touch 文件名
  • 删除文件/文件夹：
    • rm -rf 文件/文件夹
    • rm -rf /* 不要使用
    • rm -rf * 也不要使用
  • 查看文件内容：cat 文件
  • 编辑文件：vi 文件
    • 三种模式
      • 一般模式：使用 vi 命令回车之后默认进入的就是一般模式（删除指令模式中的:）
        • 特点：不能编辑文件内容，只能通过一些快捷键去操作文件内容
        • dd: 删除一行数据
        • yy: 赋值一行数据
        • p: 粘贴一行数据
        • u: 撤销操作
      • 编辑模式：自由编辑
        • 如何进入
          • 在一般模式下按 a || A || i || I || o || O
      • 指令模式：只能从一般模式进入
        • :q：退出
        • :w：保存
        • :wq：保存退出
  • Linux 上文件安装
    • 压缩包安装：.tar.gz, .tar.xz
      • .tar.gz: tar -zxvf xxx.tar.gz -C 解压的路径
      • .tar.xz: tar -xvf xxx.tar.xz -C 解压的路径
    • 使用安装包安装：.rpm
      • 查看包
        • rpm -qa | grep mysql
      • 安装包
        • rpm -ivh 包名
    • 使用软件商店安装: yum
      • yum install -y 软件名
      • yum update 软件名
      • yum remove 软件名
  • Windows上复制一个文件到Linux服务器
    • scp -r 本地需要上传的文件路径 root@192.168.100.80:/opt
  • 配置环境变量 vim /etc/profile
    • 进入文件之后光标移动到最后一行 按o开始配置 export PATH=$PATH
    • 环境变量配置完成之后需要执行如下命令把配置文件进行加载
